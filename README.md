## 📂 项目列表

### 1. [ARP Spy Tool (局域网瑞士军刀)](spy_tool/README.md)

一个基于 Linux 的轻量级网络攻防工具，利用 ARP 欺骗原理对局域网目标进行控制。

- **核心功能**：
    - 🚫 **断网模式**：强制阻断目标设备连接互联网。
    - 🕵️ **监听模式**：开启流量转发，嗅探 HTTP URL 和 DNS 查询。
- **技术特点**：
    - 自动网卡识别与 MAC 地址解析。
    - 支持高频发包压制与退出时自动恢复网络 (Re-ARP)。
    - 使用 C 语言编写，基于 `libpcap` 和 `glib`。

👉 [点击查看详细文档与使用说明](spy_tool/README.md)
